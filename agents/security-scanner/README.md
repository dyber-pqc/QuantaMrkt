# security-scanner

![PQC Native](https://img.shields.io/badge/PQC-Native-7c3aed) ![ML-DSA-87](https://img.shields.io/badge/ML--DSA-87-0284c7) ![Apache 2.0](https://img.shields.io/badge/license-Apache%202.0-green)

## What it does

`security-scanner` is a high-trust agent that pairs userspace linting
with kernel-side attestation. It walks a codebase with `pqc-lint` to
surface quantum-vulnerable primitives, then signs and verifies each
eBPF sensor it deploys via `pqc-ebpf-attestation` so only ML-DSA-87
attested programs can be loaded into the kernel.

## Tools it composes

- [`pqc-lint`](/tools/pqc-lint) - classical crypto source scanner
- [`pqc-ebpf-attestation`](/tools/pqc-ebpf-attestation) - ML-DSA load gate for kernel eBPF programs

## Quick start

```bash
pip install pqc-agent-security-scanner
python agent.py
```

## Identity

The DID from `identity.json` is a placeholder. The real ML-DSA-87
keypair is generated by the `seed-agents.yml` GitHub Action using
`liboqs`, then platform-signed with the `did:web:quantamrkt.com`
ML-DSA-87 root key before being written into this file.
