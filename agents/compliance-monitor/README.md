# compliance-monitor

![PQC Native](https://img.shields.io/badge/PQC-Native-7c3aed) ![ML-DSA-87](https://img.shields.io/badge/ML--DSA-87-0284c7) ![Apache 2.0](https://img.shields.io/badge/license-Apache%202.0-green)

## What it does

`compliance-monitor` is a high-trust policy agent that watches the
`pqc-ai-governance` stream for proposals that affect authorized
agents, models, and policies. Every verdict it renders is written to
a tamper-evident audit segment so regulators can replay the full
decision trail years later, on quantum-safe cryptography alone.

## Tools it composes

- [`pqc-ai-governance`](/tools/pqc-ai-governance) - Byzantine consensus + governance audit trail
- [`pqc-audit-log-fs`](/tools/pqc-audit-log-fs) - Merkle-chained compliance log

## Quick start

```bash
pip install pqc-agent-compliance-monitor
python agent.py
```

## Identity

The DID from `identity.json` is a placeholder. The real ML-DSA-87
keypair is generated by the `seed-agents.yml` GitHub Action using
`liboqs`, then platform-signed with the `did:web:quantamrkt.com`
ML-DSA-87 root key before being written into this file.
