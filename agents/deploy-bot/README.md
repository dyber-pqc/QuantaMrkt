## What it does

`deploy-bot` is the CD agent for PQC-secured edge appliances. It
packages firmware with an ML-DSA signature, verifies its own signature
against the delivered bytes before rollout, and writes every deployment
event to an immutable audit segment. Edge devices refuse to boot
anything that isn't signed by this agent's DID.

## Tools it composes

- [`pqc-bootloader`](/tools/pqc-bootloader) - quantum-safe signed-boot for AI appliances
- [`pqc-audit-log-fs`](/tools/pqc-audit-log-fs) - Merkle-chained deployment log

## Quick start

```bash
pip install pqc-agent-deploy-bot
python agent.py
```

## Identity

The DID from `identity.json` is a placeholder. The real ML-DSA-65
keypair is generated by the `seed-agents.yml` GitHub Action using
`liboqs`, then platform-signed with the `did:web:quantamrkt.com`
ML-DSA-87 root key before being written into this file.
