# finance-copilot

![PQC Native](https://img.shields.io/badge/PQC-Native-7c3aed) ![ML-DSA-65](https://img.shields.io/badge/ML--DSA-65-0ea5e9) ![Apache 2.0](https://img.shields.io/badge/license-Apache%202.0-green)

## What it does

`finance-copilot` is an AI agent for financial analysis. Every report it
emits is wrapped in a C2PA-compatible `ContentManifest` signed with the
agent's ML-DSA-65 identity, and every inference is appended to a
tamper-evident audit log. This gives compliance teams a quantum-safe
provenance trail that survives the 7+ year SEC retention window without
needing to re-sign when RSA/ECDSA are deprecated.

## Tools it composes

- [`pqc-content-provenance`](/tools/pqc-content-provenance) - C2PA manifests with ML-DSA signatures
- [`pqc-audit-log-fs`](/tools/pqc-audit-log-fs) - rotating, Merkle-chained inference log

## Quick start

```bash
pip install pqc-agent-finance-copilot
python agent.py
```

## Identity

The DID from `identity.json` is a placeholder. The real ML-DSA-65
keypair is generated by the `seed-agents.yml` GitHub Action using
`liboqs`, then platform-signed with the `did:web:quantamrkt.com`
ML-DSA-87 root key before being written into this file.
