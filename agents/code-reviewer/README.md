## What it does

`code-reviewer` is a CI-grade AI agent that reviews pull requests.
It scans the diff with `pqc-lint` to flag classical crypto (RSA, ECDSA,
DH, MD5, SHA-1) and posts its verdict back to the CI orchestrator over
the `pqc-mcp-transport` channel, with every JSON-RPC message wrapped
in an ML-DSA-65 `_pqc` envelope so the orchestrator can prove *this*
agent really produced *this* review.

## Tools it composes

- [`pqc-mcp-transport`](/tools/pqc-mcp-transport) - ML-DSA-signed Model Context Protocol messages
- [`pqc-lint`](/tools/pqc-lint) - detects quantum-vulnerable crypto in source code

## Quick start

```bash
pip install pqc-agent-code-reviewer
python agent.py
```

## Identity

The DID from `identity.json` is a placeholder. The real ML-DSA-65
keypair is generated by the `seed-agents.yml` GitHub Action using
`liboqs`, then platform-signed with the `did:web:quantamrkt.com`
ML-DSA-87 root key before being written into this file.
